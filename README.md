File Explorer Emulator - made for my Operating Systems classes at the sixth semester of college.
Made with Rust and the Dioxus framework.

-- Keyboard Commands:

Ctrl + N = New

Ctrl + D = Delete

Ctrl + R = Rename

Ctrl + C = Copy

Ctrl + V = Paste

Ctrl + X = Cut

Ctrl + Equal = Images Layout

Ctrl + Minus = Table Layout
